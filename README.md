# **Black Friday Sales Analysis**

This project explores Black Friday sales data to understand how customer demographics and product categories affect purchase behavior.

## **Dataset**

- CSV file (~23 MB) with 537,000+ rows and 12 columns. 
- Key columns: User_ID, Product_ID, Gender, Age, Occupation, City_Category, Stay_In_Current_City_Years, Marital_Status, Product_Category_1/2/3, Purchase.

## **Goals**

- Clean and understand the dataset structure.  
- Analyze how gender, age, marital status, and occupation impact purchases.  
- Study product category preferences and multi‑column relationships.

## **Main Analyses**

- Data overview: Missing values, data types, unique values.  
- Gender analysis: Distribution of customers and total purchase by gender (male customers are more frequent in this dataset). 
- Age & marital status: Spending patterns across age groups and marital status.  
- Multi‑column analysis: Combine age, gender, and marital status to see detailed patterns.  
- Occupation & products: Popular product categories within each occupation.

## **Tech Stack**

- Python, pandas, numpy  
- seaborn, matplotlib  
- Jupyter Notebook

## **How to Run**

1. Clone the repo and install requirements.  
2. Download the Black Friday CSV and place it in the project folder.  
3. Open the notebook(s) and run the cells to reproduce the analysis.
